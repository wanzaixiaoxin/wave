# 🚗 造物运输大亨 — 游戏需求文档 v2.0（养成版）

> **游戏定位**：交通工具养成 × 经济模拟 × 科技树升级  
> **核心体验**：亲手打造、培养每一辆车，从独轮车到星际飞船，每辆都是你的宝贝  
> **一句话**：造车→养车→进化→赚钱→造更好的车→养更好的车

---

## 一、核心理念：养成 × 经济双螺旋

### 1.1 这不是工厂，是"育车场"

```
    ❌ 旧思维                    ✅ 新思维
    ─────────────────────────────────────────
    造车 → 卖钱                   造车 → 培养 → 进化
    车是消耗品                   每辆车都是独立的伙伴
    只关心收入和效率             关心车的成长和个性
    低级车=垃圾，直接跳过       低级车也有养成上限和价值
    量产量销                     精养精售
```

### 1.2 核心养成循环

```
① 建造一辆新车 🛴
      ↓
② 给它取名字、陪它跑订单 → 获得经验值 📈
      ↓
③ 升级 → 解锁新技能/外观 ✨
      ↓
④ 进化 → 形态蜕变，收入暴增 🌟
      ↓
⑤ 成为王牌车辆 → 放入"荣誉殿堂" 🏆
      ↓
⑥ 用赚来的钱解锁下一代车型 → 继续养新的伙伴
```

### 1.3 双轨快乐

```
养成线（情感驱动）          经济线（策略驱动）
───────────────              ───────────────
为爱车升级而开心            收入增长有成就感
收集不同个性的车            解锁新科技线的爽感
看着车从弱变强              工厂规模化的满足
每辆车都有回忆              数值最优化的策略
```

## 二、核心养成系统

### 2.1 每辆车都是一个独立个体

每个车辆不再是数字叠加，而是拥有独立身份的实体：

| 属性 | 说明 | 养成方式 |
|------|------|----------|
| 🆔 **编号** | 例如"你造的第 7 辆小汽车" | 自动生成 |
| ✏️ **昵称** | 玩家可自定义 | 手动命名 |
| 📊 **等级** | 1-10 级，决定基础属性倍率 | 跑单/训练获得经验 |
| 📈 **经验值** | 每完成一次订单/运输获得 | 持续累积 |
| ⭐ **品质** | 白→蓝→金，影响成长上限 | 培养 + 材料 |
| 🧬 **性格特质** | 随机生成 1 个，影响行为表现 | 随机 + 遗传 |
| 🎨 **外观** | 可解锁涂装/装饰 | 成就/活动 |
| 💖 **亲密度** | 与玩家的羁绊等级 | 互动/陪伴 |
| 📜 **战绩** | 运输总里程、总收入等荣誉记录 | 自动记录 |

### 2.2 等级与经验系统

**经验获取公式：**
```
每次完成获得经验 = 该次收入 × (0.5 + 品质系数)
```
- 白品质：品质系数 = 0
- 蓝品质：品质系数 = 0.5
- 金品质：品质系数 = 1.5

**每级所需经验 = 50 × 1.5^(等级-1)**

| 等级 | 称号 | 解锁内容 |
|------|------|----------|
| Lv.1-3 | 🟢 新手 | 基础属性升级 |
| Lv.4-6 | 🔵 熟手 | 解锁第 1 个技能槽 + 外观自定义 |
| Lv.7-9 | 🟣 精英 | 解锁第 2 个技能槽 + 专属订单 |
| Lv.10 | 🔴 传说 | 可进化 + 进入荣誉殿堂 + 全厂加成 |

### 2.3 品质系统（核心养成目标）

**品质提升 = 核心养成投入点**

| 品质 | 升级条件 | 效果 |
|------|----------|------|
| ⚪ **白板** | 初始 | 等级上限 5，无额外加成 |
| 🔵 **精良** | 跑完 10 单 + 200 金币 + 零件×5 | 等级上限 8，经验获取 +50% |
| 🟡 **传说** | 等级 7 + 1,000 金币 + 零件×30 | 等级上限 10，收入倍率 ×2，可进化 |

**设计意图**：品质从白→蓝→金三级递进，每一级都是质变而不是量变。投入资源从白到蓝需要"跑一阵子"，从蓝到金需要"认真养它"——养成分流依然存在，但更清晰。

### 2.4 性格特质系统

每辆车出厂时随机获得 **1 个性格特质**，特质只增不减，让每辆车都有清晰定位。部分特质有稀有度区分：

| 特质 | 稀有度 | 效果 |
|------|--------|------|
| 🏎️ **勤快** | 普通 | 订单完成速度 +15% |
| 💪 **强壮** | 普通 | 载货量 +20% |
| 🎯 **精准** | 普通 | 暴击率 +5% |
| 🧠 **聪明** | 普通 | 获得的经验 +20% |
| 💰 **招财** | 普通 | 收入 +10% |
| 🍀 **幸运** | 🔥 稀有（15%） | 暴击时 ×3 而不是 ×2 |

**设计意图**：6 种纯增益特质，降低理解成本。普通特质 5 种概率均等，稀有特质 "幸运" 只有 15% 概率——玩家刷到会截图分享，天然传播素材。后续版本可追加更多特质。

## 三、进化系统（核心爽点）

### 3.1 什么叫"进化"？

当一辆车达到**满级 + 满品质**后，可以进行**进化**——形态发生质变，变成该车型的进阶版本，收入和能力大幅提升。

```
进化前                   进化后
──────────────────────────────────────
🛴 独轮车       →   🛴 涡轮独轮车（加装小引擎）
🚲 自行车       →   🚲 电动自行车（加电池+电机）
🐴 马车         →   🐴 豪华马车（镀金+弹簧悬挂）
🚗 小汽车       →   🚗 跑车（流线型车身+涡轮增压）
🚛 卡车         →   🚛 擎天柱卡车（重型武装）
🚂 火车         →   🚂 磁悬浮列车（悬浮轨道）
🚢 轮船         →   🚢 豪华邮轮（多层甲板）
✈️ 飞机         →   ✈️ 超音速客机（突破音障）
🚀 火箭         →   🚀 可回收重型火箭（着陆腿）
🛸 星际飞船     →   🛸 超光速飞船（曲速环）
```

### 3.2 进化条件

| 条件 | 说明 |
|------|------|
| 等级达到当前品质上限 | 传说品质需 Lv.10 |
| 亲密度 ≥ 80 | 需要日常互动 |
| 进化材料 | 消耗大量零件 + 金币 + 特定科技 |
| 进化任务 | 完成一系列专属订单考验 |

### 3.3 进化效果

| 项目 | 进化前 | 进化后 |
|------|--------|--------|
| 基础收入 | 100% | 300% |
| 等级上限 | 10 | 15（突破上限） |
| 技能槽 | 2 | 4 |
| 外观 | 基础款 | 专属进化皮肤 |
| 全厂加成 | 无 | 同型号产量 +10% |

**设计意图**：进化是养成的终极回报，是情感和策略的双重高潮。进化后的车辆不仅是收入主力，更是玩家身份的象征。

---

### 3.4 车型天赋树（每款车都有专属定位）

进化后的每款车型获得一个**专属天赋**，让不同车型不只是换皮：

| 车型 | 进化名 | 专属天赋 | 改变玩法 |
|------|--------|----------|----------|
| 🛴 独轮车 | 涡轮独轮车 | 🎯 灵活机动：订单完成速度 +20% | 更快周转 |
| 🚲 自行车 | 电动自行车 | 🔋 持久续航：可连续接 2 单 | 翻倍效率 |
| 🐴 马车 | 豪华马车 | 👑 贵族气质：高品质订单优先派给此车 | 稳定高收入 |
| 🚗 小汽车 | 跑车 | 🏎️ 极速狂飙：短途订单收入 ×2 | 爆发型 |
| 🚛 卡车 | 擎天柱 | 📦 巨量运输：单次收入 +50% | 批量收益 |
| 🚂 火车 | 磁悬浮 | 🚄 车队联动：同型车每多 1 辆 +5% 收入 | 集群效应 |
| 🚢 轮船 | 豪华邮轮 | 🛳️ 探索发现：每次订单额外获得零件 | 资源型 |
| ✈️ 飞机 | 超音速 | 🌍 全球网络：所有订单刷新速度 +30% | 辅助全厂 |
| 🚀 火箭 | 重型火箭 | 🌙 星际物流：零件产出 +50% | 生产资料 |
| 🛸 星际飞船 | 超光速 | ✨ 曲速引擎：全车型收入 +15%（可叠加） | 终局光环 |

**设计意图**：玩家选择养什么车不再只是"挑属性"，而是根据当下的策略需求——缺零件就养轮船，想冲收入就养卡车，玩收集就养火车。每辆车都有自己的"生态位"。

| 方式 | 亲密度增量 | 冷却 |

## 五、重组养成后的经济系统

### 5.1 收入模式重塑

#### 收入来源改为"派单制"

不再是无感情地"生产→自动卖"，而是：

```
造车 → 车辆去跑订单 → 按能力和等级赚取收入 → 车获得经验
                                                    ↓
                                          收入用来造更好的车/养车
```

**订单种类**（与养成深度绑定）：

| 订单类型 | 匹配逻辑 | 收益规则 |
|----------|----------|----------|
| 📮 **普通配送** | 任意车可接 | 收入 = 基础价 × (1 + 等级 × 0.05) |
| 🏔️ **长途运输** | 需要耐久 ≥ 3 的车 | 收入 ×2，额外大量经验 |
| 💎 **贵重物品** | 需要蓝品质以上的车 | 收入 ×3，品质越高越稳 |

> 注：后两种订单需要对应属性达标才会刷新，鼓励玩家多元化培养。

### 5.2 养成的资源投入

玩家赚的钱需要分配到两个方向，形成**养成 vs 扩张**的策略权衡：

```
      收入
     /    \
    /      \
   ↓        ↓
养车投入    扩张投入
(升级/品质/   (新车型/产线/
 进化/外观)    科技研发)
```

**关键设计**：养车投入直接提升单车的收入能力，扩张投入提升整体产能和车型解锁。玩家需要自己找到平衡点。

### 5.3 养了旧车就不想换新车？—— 平滑过渡设计

这是养成游戏最大的设计难题："我都把独轮车养到 10 级了，为什么要换自行车？"

解决方案 —— **三重激励机制**：

#### 机制 A：新车体验加速

当你解锁下一 tier 车型时，该车型**前 5 次订单获得 3 倍经验 + 2 倍收入**，快速建立情感。

#### 机制 B：旧车退役不浪费

| 退役方式 | 效果 |
|----------|------|
| 🏆 **荣誉殿堂** | 满级车放入殿堂，提供全厂永久加成（+1%~5% 收入） |
| 🔄 **传承** | 旧车退役时，将 1 个性格特质遗传给新车 |
| 🔧 **拆解回收** | 回收 60% 的养成资源（零件/金币） |
| 📸 **纪念册** | 保留头像/名字/战绩在纪念册中，随时查看 |

#### 机制 C：同车型收集加成

每收集满 **3/5/10 辆同一车型的满级车**，解锁该车型的**全厂光环**：

| 收集数量 | 光环效果 |
|----------|----------|
| 3 辆满级 | 该车型生产速度 +10% |
| 5 辆满级 | 该车型收入 +15% |
| 10 辆满级 | 该车型暴击率 +5% + 专属皮肤 |

**设计意图**：旧车养满不是终点，而是为全厂做贡献的新起点。玩家可以自豪地说"看，这是我养过的 10 辆传说独轮车！"

## 六、养成型科技树

### 6.1 科技树（单线简洁版）

不再多分支选择，改为**一条主线、逐级解锁**——任何时候玩家只需关注下一级：

```
L5 ──── 星际物流 ──── 解锁 T10 星际飞船，全厂 +50% 收入
L4 ──── 全球供应链 ──── 解锁 T8-T9（飞机/火箭），产线 +1
L3 ──── 自动化产线 ──── 解锁 T6-T7（火车/轮船），生产速度 +25%
L2 ──── 内燃机 ────── 解锁 T4-T5（小汽车/卡车），品质上限 +1
L1 ──── 基础机械 ──── 解锁升级功能、品质系统
```

| 科技 | 解锁条件 | 消耗 | 效果 |
|------|----------|------|------|
| L1 基础机械 | 初始 | 100 金币 | 解锁属性升级、品质系统 |
| L2 内燃机 | 产 5 辆 T3 马车 | 800 金币 + 零件×10 | 解锁 T4-T5，品质上限 +1 |
| L3 自动化产线 | 产 5 辆 T5 卡车 | 5,000 金币 + 零件×50 | 解锁 T6-T7，生产速度 +25% |
| L4 全球供应链 | 产 3 辆 T7 轮船 | 30,000 金币 + 零件×200 | 解锁 T8-T9，产线 +1 |
| L5 星际物流 | 产 2 辆 T9 火箭 | 200,000 金币 + 零件×1,000 | 解锁 T10，全厂收入 +50% |

### 6.2 养成相关的成就进阶

| 成就 | 条件 | 奖励 |
|------|------|------|
| 🍼 **第一声啼哭** | 造出第 1 辆车 | 给它命名的权利 |
| 🛴 **飞驰人生** | 同一辆车完成 100 单 | 勋章 "老司机" |
| 🏅 **五维全能** | 同一辆车集满 5 级属性 | 专属勋章 "全能选手" |
| 💖 **至死不渝** | 与同一辆车亲密度达到 100 | 勋章 "最好的伙伴" |
| 👨‍👩‍👧‍👦 **大家族** | 同时拥有 10 辆传说品质车 | 工厂外观焕新 "星光大道" |
| 🔥 **进化狂人** | 完成第 1 次进化 | 全服公告 [玩家名] 的 [车名] 进化了！ |
| 🌈 **彩虹战队** | 集齐 7 种不同性格特质的传说车 | 专属皮肤 "彩虹战队" |

## 十、养成与经济的闭环验证

### 10.1 资源流动总图

```
                    ┌──────────────┐
                    │    🎯 订单    │
                    └──────┬───────┘
                           │ 派车接单
                           ↓
    ┌──────────────┐  ┌──────────────┐
    │  🏭 工厂造车  │→│  🚗 车辆跑单  │
    └──────────────┘  └──────┬───────┘
           ↑                  │
           │           ┌──────┴───────┐
           │           │              │
           │           ↓              ↓
           │       🪙 收入      📈 经验+亲密度
           │           │              │
           │           ↓              ↓
           │    ┌──────────┐  ┌──────────────┐
           │    │ 🔬 科技   │  │ ⬆️ 等级+品质  │
           │    │ 🏭 工厂   │  │ 🔄 进化       │
           │    └──────────┘  └──────┬───────┘
           │                         │
           └─────────────────────────┘
                (赚更多钱造更多车)     (车变强→赚更多)
```

### 10.2 为什么这个闭环不会崩

| 潜在问题 | 解决方案 |
|----------|----------|
| 只养一辆车就够了 | 每辆车只能接一个订单 → 需要多辆车并行 |
| 只养低级车高效 | 低级车收入上限低 → 高级车收入碾压 |
| 只造车不养也能玩 | 不养的车属性差+收入低 → 不划算 |
| 养车投入不如造新车 | 养成后收入 ×3 + 全厂光环 → 长期回报高 |
| 亲密度太难不想互动 | 日常操作（命名/清洗）都有收益 → 正反馈循环 |
| 进化后旧车没用了 | 荣誉殿堂 + 传承 + 纪念册 → 回忆永存 |

## 十一、心流设计：创造与分配玩家的注意力

### 11.1 心流理论基础（简版）

> **心流 = 挑战与技能的动态平衡，是游戏让人停不下来的底层密码**

```
        高
         │  焦虑 😰         心流 🎯         厌倦 😴
  挑战   │  (太难了！)   →  (刚刚好！)   →  (没意思了！)
         │
         │   ↑                    ↑                    ↑
         │  你只有 3 条产线      你有 5 辆车要养      所有车都满级了
         │  科技点永远不够       刚够解锁新科技       科技全解锁了
         │  工厂升级太贵         赚的钱刚好够升级     钱多到花不完
         │
        低
               技能/能力 →
               低                             高
```

**设计的核心任务**：让玩家始终沿着中间这条"心流通道"前进，左边是焦虑、右边是无聊。

---

### 11.2 四层心流循环（游戏的脉搏）

本游戏设计了**4 层节奏嵌套**的心流循环，每一层都有自己的反馈闭环：

```
时间尺度        循环名称        做什么                 反馈速度
────────────────────────────────────────────────────────────────
⏱️ 1-5 秒      微循环          点击车辆互动              即时
    │                          清洗/保养
    │                          车辆表情反馈
    │
⏱️ 30s-3min    浅循环          派车接单→等待→收钱      快速
    │                          属性升级
    │                          造新车
    │
⏱️ 10-30min    深循环          养一辆车满级→进化      中速
    │                          解锁新 Tier
    │                          点亮科技树分支
    │
⏱️ 2-5 小时    终局循环        T1→T10 全通关          慢速
                               全科技点亮
                               收集全车型传说
```

**关键设计**：这 4 层循环**同时运转**。玩家在任何时刻都有事可做：
- 等一辆车跑完订单（浅循环等待中）→ 可以去互动另一辆车（微循环）
- 养车到满级的途中（深循环进行中）→ 顺手造新车扩张产线（浅循环）
- 永远有短、中、长期目标在同时进行

### 11.3 心流曲线：全游戏节奏设计

```
心流强度
  │
  │    🏔️   🏔️         🏔️                🏔️        🏔️
  │   /    \ / \       / \               / \       / \
  │  /  🏔️  🏔️  \ 🏔️ /   \ 🏔️   🏔️    /   \ 🏔️ /   \ 🏔️
  │ /  /    \    \ /     \ / \ / \  🏔️ /     \ /     \ / \
  │/  /      \    /       \   🏔️  \ / /       \       /   \
  │  /        \  /         \ /      \ /         \     /     \
  │───────────────────────────────────────────────────────────▶ 时间
  │ T1   T2   T3   T4   T5   T6   T7   T8   T9   T10  通关后
  │  新手期    发展期    黄金期    冲刺期    终局期    收集期
```

**每个山峰 = 一次「进化/解锁新车型」的高潮体验**  
**每个山谷 = 积累资源、培养车辆的平淡期（但有事做）**

关键设计原则：**不让任何一段低谷超过 5 分钟**。如果玩家超过 5 分钟没有"事件"发生，他就可能流失。

分阶段心流设计：

| 阶段 | 时长 | 心流特征 | 玩家感受 | 风险点 |
|------|------|----------|----------|--------|
| 🟢 **T1-T3 新手期** | 0-5min | 快速学习，频繁解锁 | "哇还能这样？" | 教程太多→烦躁 |
| 🟡 **T4-T5 发展期** | 5-15min | 选择变多，策略初现 | "我要先升级哪个？" | 选择过多→焦虑 |
| 🟠 **T6-T7 黄金期** | 15-30min | 深度投入，心流峰值 | "太爽了一直在升级" | 没有新内容→无聊 |
| 🔴 **T8-T9 冲刺期** | 30-45min | 高挑战高回报 | "再坚持一下就到星际了！" | 太难→挫败 |
| 💜 **T10 终局期** | 45-60min | 成就感爆棚 | "我就是运输之王！" | 通关后→空虚 |
| 🌈 **通关后收集期** | ∞ | 自由探索，自我挑战 | "我要集齐所有传说车" | 目标不清晰→流失 |

### 11.4 注意力分配设计（核心：如何不让玩家手忙脚乱）

这是养成经营游戏最难的课题：**当玩家有 10 辆车、5 条产线、20 个未完成订单时，怎么让他不感到焦虑？**

#### 原则一：注意力漏斗

在任何时刻，只让玩家看到 **3-5 个需要关注的事项**，其他全部折叠或静默：

```
你的全部事务（20+项）                玩家的视野（3-5项）
┌──────────────────────┐           ┌──────────────────┐
│ 🚗 小轮子 Lv.42      │           │ ⭐ 重点关注      │
│ 🚗 闪电号 Lv.28      │  过滤    │ 🚗 小轮子 订单快完成了！│
│ 🚗 风之子 Lv.15      │  ─────→  │ 🚗 闪电号 升到Lv.30后│
│ 🚗 小黄  Lv.8        │           │    可以进化！     │
│ 🚗 伯爵  Lv.3        │           │                   │
│ 🚗 ...(还有15辆)     │           │ 📋 新订单          │
│ 🏭 工厂升级           │           │ 贵重物品运输 ×3倍！│
│ 🔬 科技等待研发       │           │                   │
│ 📦 3个待处理的订单    │           │ 🔬 建议升级        │
│ 🏆 3个接近成就        │           │ 「基因图谱」可提升 │
│                      │           │  新车特质概率      │
│  (全部显示→焦虑)     │           │  (智能聚焦→心流)  │
└──────────────────────┘           └──────────────────┘
```

**实现方式**：
- 系统自动标记 **"最佳下一步"**（用星星/光效提示）
- 快要完成的事项置顶显示（如 "小轮子 30s 后完成订单"）
- 长时间未操作的事项折叠到二级页面
- 玩家可手动标记 "关注" 和 "忽略"

#### 原则二：批次化同类操作

不要让玩家在不同类型的操作之间频繁切换。同类操作**聚集在一起**：

```
❌ 低心流设计（碎片化切换）：
  派单 → 等 5s → 点一下属性升级 → 等 10s → 切去清洗另一辆车
  → 等 3s → 切去科技树 → 等 2s → 切回派单 → 🔄 不断切，心累

✅ 高心流设计（批次化操作）：
  ① 派单时刻：一口气给 5 辆车全部派好订单
     ↓（车辆开始跑单，进入等待期）
  ② 养成时刻：等单的时候，集中给几辆车升级/清洗/保养
     ↓
  ③ 管理时刻：切到工厂/科技树做战略决策
     ↓
  ④ 收获时刻：订单完成，集中收钱 + 新成就弹窗
     ↓（回到①，循环）
```

**实现方式**：
- 车库页面：**批量派单**（勾选多辆车，一键派单）
- 养成页面：批量升级同理
- 系统提示"养成时刻！" / "收获时刻！" 引导玩家的节奏

#### 原则三：注意力权重随数量动态调整

车辆的 **UI 展示权重** 根据玩家当前状态动态变化：

| 车辆状态 | UI 展示 | 注意力权重 |
|----------|---------|-----------|
| 🆕 刚出厂 | 突出展示 "新！" | 高（引导命名+初印象） |
| ⏳ 订单进行中 | 进度条 + 剩余时间 | 中（时不时看一眼） |
| 📈 即将升级 | 闪烁箭头 | 中高（提醒去升级） |
| 🔒 可进化 | 金色光效 + 角标 | **极高**（主要目标） |
| 😴 离线中 | 缩小尺寸/灰色 | 低 |
| 🏆 已满级/已进化 | 放在荣誉殿堂 | 极低（除非查看） |

**目的**：玩家的注意力自然流向最需要被关注的对象，而不是被信息淹没。

#### 原则四：让"空闲时间"也有事做

心流的最大杀手是"等待时不知道干什么"。设计**等待时间的填充活动**：

```
订单进行中（30s 等待）
    │
    ├─ 🚿 去洗另一辆车（微循环）
    ├─ 📊 查看统计/成就进度
    ├─ 🔬 研究科技树（当前推荐：L2 内燃机）
    ├─ 🏭 调整工厂布局
    ├─ 📸 截图留念
    └─ 💬 点击车辆互动（每辆车有不同反应）
```

**关键**：填充活动必须是**可选但不强制**的。玩家可以选择主动操作，也可以放着不管等自动完成。这样"等待"变成了"自主选择"，而不是"被强制停滞"。

| 😤 **被迫等待** | 产线全满，但不想下线等 | 离线收益 + 手机通知（可选） |
| 😵 **信息过载** | 第一次看到科技树，20 个节点 | 引导解锁路径："推荐路线"高亮 |

### 11.7 不同玩家类型的心流适配

| 玩家类型 | 行为特征 | 心流设计方向 |
|----------|----------|-------------|
| 🎮 **成就型** | 追求全收集、全成就 | 收集进度条、稀有度显示、全图鉴 |
| 🧠 **策略型** | 追求最优效率、数值最大化 | 统计面板、效率排名、性价比分析 |
| 💕 **情感型** | 享受养成过程、和人/车建立连接 | 亲密度互动、性格对话、回忆功能 |
| ☕ **休闲型** | 碎片时间玩、挂机为主 | 离线收益最大化、一键操作 |
| 🔥 **竞技型** | 想和别人比 | 排行榜、稀有度展示、分享功能 |
| 📖 **收集型** | 集齐所有性格/车型/成就 | 全图鉴进度、稀有度提示、限时活动 |

**设计目标**：游戏不强制玩家类型。一个情感型玩家可以完全不关心效率，一个策略型玩家也可以跳过互动。**所有玩法路径都能通关**。

---

### 11.8 心流分配的核心公式

```
注意力分配 = 车辆数量 × 每车关注度 × 时间切片
```

当车辆数量增长时：
- **车辆少（1-3 辆）** → 每车关注度高，深度养成 → 心流来自"深度连接"
- **车辆中（4-8 辆）** → 每车关注度中等，策略分配 → 心流来自"优化资源分配"
- **车辆多（9+ 辆）** → 每车关注度低，系统化管理 → 心流来自"构建高效系统"

**公式在 UI 层面的实现**：

```
1-3 辆车 → 大卡片展示，每辆占 1/3 屏幕
           点击查看详情（深度互动）

4-8 辆车 → 中等卡片，可滑动浏览
           列表+筛选（效率优先）

9+ 辆车 → 缩略网格 + 智能排序
           批量操作（管理导向）
```

随着工厂扩张、车辆增多，UI 自动缩放，始终让玩家处于 **"信息量刚好"** 的状态。

---

### 11.9 心流设计清单（上线前自查）

```
□ 玩家上线后 5 秒内能否找到第一个操作目标？
□ 有没有超过 30 秒没有任何正向反馈的场景？
□ 信息量是否随玩家水平渐进增长？
□ 是否有"最佳下一步"的隐式引导？
□ 等待期间是否有填充活动？
□ 不同玩家类型是否能找到自己的乐趣？
□ 每个操作是否有清晰可见的结果？
□ 短期目标（30s）→ 中期目标（10min）→ 长期目标（1h+）是否三级齐全？
□ 焦虑点是否可被识别和缓解？
□ 无聊点是否可被识别和填补？
□ 通关后是否有新的心流目标？
□ 手机端碎片化游玩和 PC 端沉浸游玩的节奏是否都适配？
```

## 十二、自然的边界：给成长加点小曲折

### 12.1 设计的核心原则：发展 > 制约

你玩这个游戏是为了**养车**，不是为了**被管教**。

```
体验配比（100%）：
┌────────────────────────────────────────────────┐
│   🚗 养车·赚钱·解锁新玩具                      │  80%
│      (发展本身的乐趣)                           │
├────────────────────────────────────────────────┤
│   ⚖️ 偶尔做点小取舍 + 😢 告别老朋友            │  20%
│      (边界带来的滋味)                           │
└────────────────────────────────────────────────┘
```

**三条铁律**：
1. **制约不能盖过发展**——80% 的时间玩家在感受"变强了"，20% 在感受"需要取舍了"
2. **早期不制约**——T1-T4（前 15 分钟）没有任何边界，让玩家专心享受养成
3. **制约是温柔的**——不用"警报警报"式的 UI，而是"温馨提示"式的语气

---

### 12.2 只有一个真正的边界：车库空间

与其搞 4 套系统让玩家手忙脚乱，不如只保留**一个最自然的边界——车库空间**。

```
为什么只保留空间？

✅ 直觉的——"车库满了"谁都懂，不需要学
✅ 有情感的——"要腾位置就得送走老朋友"
✅ 有取舍的——"留哪些车在队里"是真实的选择
✅ 有节奏的——满了→取舍→空出来→再养新的
✅ 不焦虑的——满了不会惩罚你，只是不能造新车了

❌ 市场供需 → 太抽象，玩家需要算 "我怎么知道市场在想什么？"
❌ 生态足迹 → 太沉重，玩个游戏还要被道德审判
❌ 车辆寿命 → 有意义但太伤感，让其融入空间取舍的自然结果
```

#### 车库空间具体规则

```
你有一个车库，初始容量 6 辆车。

• 独轮车/自行车/马车  → 占 1 格
• 小汽车/卡车/火车    → 占 2 格
• 轮船/飞机           → 占 3 格
• 火箭/星际飞船       → 占 4 格
```

**满了之后**：

```
车库满了 ≠ 世界末日

┌──────────────────────────────────────┐
│ 🚗 你的车库 (6/6 已满 🈵)           │
│                                      │
│ 想造新车？需要先腾个位置出来：        │
│                                      │
│ ① 🏠 扩建车库 → 花金币+等一会儿     │
│     "地方大了，能养更多孩子"          │
│                                      │
│ ② 📤 把一辆车送走 → 放入荣誉殿堂    │
│     "你陪我的日子，我不会忘记"        │
│     (该车永久保留名字/战绩在纪念册中) │
│                                      │
│ ③ 🔧 拆解回收 → 获得零件            │
│     "你的零件会在新车里重生"          │
│     (概率传承特质)                   │
│                                      │
│ (没有惩罚选项，没有焦虑选项)          │
└──────────────────────────────────────┘
```

**车库扩建**：

```
扩建车库：
• 每次 +2 个车位
• 花费 ≈ 当前 tier 基准价 × 5
• 扩建时有 10s 施工动画（看着车库变大）
• 上限 = 12 个车位（足够终局使用）
```

**设计意图**：车库空间像"朋友圈人数上限"——满了就不得不做些取舍，但这取舍是有温度的，不是惩罚。

---

### 12.3 空间的自然副产品：告别也是养成的一部分

空间边界最有价值的地方在于——它让**告别**成为自然发生的事，而不是强制的。

```
没有空间边界时：
    一直造车 → 一直存着 → 车越来越多 → 但没有一辆是特别珍视的
    (没有稀缺，就没有珍惜)

有空间边界时：
    车库满了 → 你看着每一辆车 → "这辆陪我最久……这辆是我第一辆传说……"
    → 你做出选择 → 送走的那一辆会永远留在纪念册里
    → 留下的那一辆，你会更珍惜它
    (因为你知道，它也在倒计时)
```

**情感流程**：

```
你看着车库里的 6 辆车，需要送走一辆来腾位置。

你的独轮车"小轮子"——你的第一辆车，Lv.42，品质传说。
它已经很老了，订单效率不如新车。
但它陪你走过了最开始的艰难岁月。

你点开它的页面，看了很久。
→ "小轮子，谢谢你。"

你选择了"放入荣誉殿堂"。
系统弹出一段动画：小轮子缓缓驶入一个灯光璀璨的展厅，
你的第一辆车——永远在那里。

你获得一枚勋章：「最初的伙伴」
空出来的位置，你用来造一辆新的小汽车。
新车出厂时，有一定概率继承小轮子的一个特质。
```

这比任何抽象的系统都更有温度。

---

### 12.4 车辆的隐式生命周期（不告诉玩家的"寿命"）

**不搞明确的健康度数值条**，而是通过**隐式信号**让玩家自然感受到车辆变老了：

```
显式信号（玩家看到）                   隐式机制（背后运作）
────────────────────────────────────────────────
• 车辆页面出现小小的"怀念"标签           • 每完成 200 单，触发一次"退化"
• 偶尔跑单回来对话框多一句                 • 退化：属性 -5%，但经验获取 +10%
  "今天路上有点累了……"                    （越是老车，越有智慧）
• 维护费逐渐增加（很慢）                  • 维护费每 100 单 +10%
• 亲密度达到上限后变成                    • 但亲密度互动有概率触发"回忆对话"
  "羁绊"状态（不能再升了）                  → 获得一次性经验奖励
```

**关键**：没有强制报废。你想留着一辆车一辈子也可以，只是它越来越慢、越来越贵——直到你自己觉得"是时候了"。

当一辆车累计完成 **500 单** 时，系统会弹出一个温柔的提示：

```
"闪电号已经陪你跑了 500 单了。
 它见证了你从一个手工作坊到如今的大工厂。
 或许有一天，它会想要休息……

 但今天不是那一天。继续跑吧，老伙计。"
```

**没有强制，只有仪式感。**

---

### 12.5 制约的节奏：只在适当的时机出现

```
游戏进程       制约登场         玩家此刻的感受
────────────────────────────────────────────────
T1 🛴       无             "造车好开心！命名！"
T2 🚲       无             "新车型！属性升级！"
T3 🐴       无             "看到它在成长，好有成就感"

T4 🚗   提示"车库快满了"    "啊已经 5 辆车了……"
         (温和提醒)          "不过我可以选择送走谁"

T5-T6      第一次需要       "有点舍不得，但为了新车……"
        做取舍              "我把它放荣誉殿堂了，永远纪念"

T7-T8      第一次送走       "它陪了我好久……没有它就没有今天的我"
        满级车              "传承了！它的特质在新车上了！"

T9-T10   自如地管理         "我已经习惯了分合"
        车库流转            "每辆车都是故事，但故事会延续"
```

**核心**：制约**从不提前出现**，也**从不突然出现**。玩家有充足的时间适应和准备。

---

### 12.6 可选深层挑战（给想要更多策略的玩家）

对于觉得"一个边界不够"的硬核玩家，在通关后或者设置中提供**可选挑战**：

```
⚙️ 设置 → 挑战模式

□ 市场波动：开启后，每种车型价格随供需浮动
  (适合想优化经济策略的玩家)

□ 环保意识：开启后，高排放车辆会触发社区事件
  (适合想要额外层层的玩家)

□ 硬核退役：开启后，车辆到达寿命上限后自动退役
  (适合想体验完整生命周期的玩家)

* 所有挑战模式默认关闭
* 可随时开启/关闭
* 不影响主线和成就
```

这样，**新手不会被吓到，老手有深度可挖**。

---

### 12.7 总结：制约与发展的配比

```
发展（80%）
┌─────────────────────────────────────────────────────┐
│  🎉 造新车 → 取名 → 跑单升级 → 进化 → 更强！      │
│  🎉 解锁新科技 → 解锁新车型 → 更多收入               │
│  🎉 工厂扩大 → 产线增多 → 更效率                     │
│  🎉 收集不同性格的车 → 打造传说车队                   │
└─────────────────────────────────────────────────────┘
边界（20%）
┌─────────────────────────────────────────────────────┐
│  🤔 车库满了 → 选择送走哪辆 → 纪念                   │
│  😢 老伙计跑不动了 → 让它光荣退役 → 传承             │
│  📐 扩建要花钱 → 是先升级科技还是先扩建？             │
└─────────────────────────────────────────────────────┘
```

**边界不是对手，是佐料。** 加了佐料的菜更好吃，但你不会为了吃佐料而点菜。

### 13.2 社交轻量化

不在 v1.0 做重度社交，预留三个轻度社交触点：

```
① 进化全服公告（成本：一行推送代码）
   → "【系统】玩家 小明 的 小轮子 进化成了 涡轮独轮车！"
   → 满足炫耀心理，刺激其他人

② 纪念册分享（成本：生成一张截图）
   → "这是我第一辆车，它叫小轮子，陪我跑了 100 单。"
   → 天然情感传播素材

③ 好友拜访（成本：中，v1.1+）
   → 可参观好友的车库，给好友的车「点赞」
   → 轻度社交，不强制，不构成压力
```

### 13.3 商业化设计

```
商业化节奏（与用户生命周期匹配）：
────────────────────────────────

Day 1 看到：首充礼包（6元）
  内容：限定涂装「金色传说」+ 200 零件
  目的：建立付费习惯，不卖数值

Day 3 看到：车库扩容卡（12元）
  内容：永久 +2 车位
  目的：空间制约恰好在这个时间点出现，玩家有付费动机

Day 7 看到：特质保底券（18元）
  内容：下次新车必出稀有特质
  目的：收集型玩家在 Day 7 开始追求稀有

Day 14+ 看到：月度通行证（30元）
  内容：每日登录领零件/涂装/限定时装
  目的：提升 30 日留存

不做：
  ❌ 不卖金币/零件（不卖数值）
  ❌ 不卖「抽车」（车必须自己造才有感情）
  ❌ 不卖体力/离线收益（充分尊重玩家时间）
```
