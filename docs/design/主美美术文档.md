# 造物运输大亨 — 主美美术文档 v1.0

> **本文档定位**：美术设计的完整规范，涵盖风格指南、车辆设计、UI 规范、动画特效。  
> **前置文档**：`主策划总纲.md`（所有数值和规则以总纲为准）  
> **渲染风格**：2D 扁平插画风 + 柔和渐变，参考《旅行青蛙》的温暖感 + 开罗游戏的像素级细节

## 第一章：美术风格总纲

### 1.1 风格关键词

```
温暖    圆润    明亮    轻量
亲近    柔和    有质感    不油腻
```

**参考作品**：
- 《旅行青蛙》—— UI 的温暖感、木材/布纹材质
- 《开罗系列》—— 信息密度高但清晰的卡片布局
- 《Stardew Valley》—— 色彩饱和度控制、柔和阴影
- 《企鹅岛》—— 车辆/动物的 Q 版表现

### 1.2 色彩体系

```
主色：#2C3E50（深蓝灰）—— 文字和主要 UI 元素
辅色：#3498DB（明亮蓝）—— 交互元素、按钮
强调色：#F39C12（暖橙）—— 重要提示、金币相关
背景色：#ECF0F1（浅灰白）—— 主背景
卡片底色：#FFFFFF + 4% 阴影—— 车辆卡片

车辆品质色：
  白板：#BDC3C7 灰色边框
  精良：#2980B9 蓝色边框 + 微光
  传说：#F1C40F 金色边框 + 光晕

亲密度色：
  0-20:  #95A5A6 灰
  21-60: #2ECC71 绿  
  61-100:#E74C3C 粉红

性格特质色（按稀有度）：
  普通：#3498DB 蓝
  稀有：#9B59B6 紫
```

## 第二章：车辆设计规范

### 2.1 车辆设计总要求

```
• 每辆车 2 个形态：基础形态 + 进化形态
• 侧视图（朝右），2D 扁平插画
• 圆润 Q 版风格，头身比约 1:1
• 带"眼睛"表情（让车有生命感）
• 每种车用颜色/配饰区分性格特质
• 设计尺寸：200×120px（基础），可缩放
```

### 2.3 性格特质视觉表现

```
每种性格特质通过以下方式体现在车辆外观上：

🏎️ 勤快   → 车身上有"汗滴"贴纸 / 速度线特效
💪 强壮   → 车身更厚实、轮胎更大、有肌肉线条
🎯 精准   → 车头有瞄准器图标 / 干净利落的线条
🧠 聪明   → 车身有"博士帽"贴纸 / 眼镜装饰
💰 招财   → 金色元素更多 / 车尾有金币拖尾粒子
🍀 幸运   → 四叶草贴纸 / 彩虹色反光

表现方式优先级：贴纸 > 颜色变化 > 粒子特效
（尽量用贴纸完成，减少额外粒子性能消耗）
```

## 第六章：资源交付规范

### 6.1 文件命名规则

```
{类型}_{子类型}_{编号}_{状态}.png

示例：
vehicle_t1_base_01.png          — 独轮车基础形态
vehicle_t1_evolved_01.png       — 独轮车进化形态
icon_gold_01.png                — 金币图标
bg_garage_01.png                — 车库背景
btn_primary_normal.png          — 主按钮默认态
btn_primary_pressed.png         — 主按钮按下态
card_vehicle_blue_border.png    — 蓝色品质卡片边框
particle_gold_01.png            — 金币粒子
```

### 6.2 输出格式

```
UI 元素：
  • 格式：PNG-32（带透明通道）
  • 尺寸：@2x 输出（以 1x 为基准设计）
  • 建议：使用 SVG 源文件，运行时缩放

车辆设计：
  • 格式：PNG-32（带透明通道）
  • 尺寸：200×120px (@1x) / 400×240px (@2x)
  • 建议：每辆车独立文件

背景/大图：
  • 格式：JPEG 85% 质量（无透明需求时）
  • 尺寸：按最大屏幕尺寸 @2x

动画帧：
  • 格式：PNG 序列（导出为 spritesheet）
  • 或者 Lottie JSON（推荐，文件更小）
```

### 6.3 交付检查清单

```
□ 所有资源已按命名规则重命名
□ 所有资源已去除非透明背景
```
