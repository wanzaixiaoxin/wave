## 第一章：未细化模块完善

### 1.1 挑战模式（完整设计）

#### 1.1.1 定位

挑战模式是通关后的主要长线内容。所有挑战模式**默认锁定**，通关 T10 后一次性解锁。不影响主线进度。

#### 1.1.2 挑战一览

| 模式 | 名称 | 核心规则 | 时长 | 难度 | 解锁条件 |
|------|------|----------|------|------|----------|
| 🅰 | **竞速挑战** | 3 分钟内完成尽可能多的订单 | 3min | 随段位提升 | T10 通关 |
| 🅱 | **生存挑战** | 车库不可扩建，极限管理 | 不限时 | 高 | 竞速挑战黄金段位 |
| 🅲 | **随机挑战** | 每局随机 1-3 个 modifier | 5min | 随机 | T10 通关 |

#### 1.1.3 竞速挑战详细规则

```
目标：3 分钟内获取最高收入
初始状态：系统分配 3 辆随机车（品质、等级、特质全随机）
        + 1 条产线
        + 1,000 起始金币

操作限制：
• 可造新车（消耗金币，价格同主线）
• 不可使用已解锁的科技（统一使用 L1）
• 不可扩建车库（固定 6 格）
• 不可使用道具/涂装（公平竞技）

结束时收入 = 最终结算金币
段位奖励：
  青铜    < 5,000      → 5 零件
  白银    5,000-15,000 → 15 零件
  黄金    15,000-30,000 → 30 零件 + 限时涂装(3天)
  钻石    30,000-50,000 → 50 零件 + 限时涂装(7天)
  传说    > 50,000      → 100 零件 + 永久涂装

冷却：每日免费 3 次，之后消耗 1 张挑战券（每日签到送 1 张）
```

#### 1.1.4 生存挑战详细规则

```
目标：在有限空间内达到最高收入
初始状态：6 格车库（不可扩建），1 条产线，5,000 金币
特殊规则：
• ❌ 不可扩建车库
• ✅ 可拆解车辆获得部分金币
• ✅ 车辆达到 Lv.10 后必须拆解或放入殿堂（殿堂在本模式中不占格）
• ✅ 科技树正常解锁（消耗降低 50%）
• 每拆解一辆车，下一次同车型建造费用 +10%（累加，上限 +100%）

通关条件：解锁 T10 并收入累计达到 500,000
奖励：永久称号「生存大师」+ 专属车库皮肤「废墟工业」

设计意图：资源极度受限下的管理策略，最优解不是"全养"而是"精养+快周转"
```

#### 1.1.5 随机挑战详细规则

```
目标：5 分钟内达到最高收入
初始状态：同竞速挑战

每局系统随机抽取 1-3 个 Modifier（互斥组合有保护逻辑）：

┌──────────────┬──────────────────────┬─────────────────────┐
│ Modifier     │ 效果                  │ 策略应对             │
├──────────────┼──────────────────────┼─────────────────────┤
│ 通货膨胀     │ 所有车辆售价 ×2       │ 快速量产，薄利多销    │
│              │ 但建造成本 ×3         │                     │
│ 罢工日       │ 产线速度 -50%         │ 提前囤车，减少依赖    │
│ 品质狂热     │ 金品质概率 ×3         │ 集中资源养一辆       │
│ 零件荒       │ 零件获取归零          │ 避免使用零件的策略    │
│ 丰收季节     │ 所有订单收入 ×3       │ 最大化派单频率       │
│ 时光飞逝     │ 车辆老化速度 ×3       │ 用完即拆，不养长期    │
│ 怀旧情怀     │ T1-T3 收入 ×5        │ 低级车当主力         │
│              │ T4+ 收入 ×0.5        │                     │
│ 独角戏       │ 车库只有 3 格         │ 极致精养             │
│ 暴风雪       │ 订单刷新速度 -70%     │ 每个订单都不放过      │
└──────────────┴──────────────────────┴─────────────────────┘

互斥保护：不会同时出现「通货膨胀+零件荒」这样完全矛盾的组合
奖励：每完成一次获得随机奖励（零件/涂装碎片/挑战券）
```

### 1.2 轮回/声望系统（完整设计）

#### 1.2.1 定位

完成 T10 通关后开启的系统，解决"通关后空虚"的问题。

#### 1.2.2 核心机制

```
玩家达到 T10 并累计收入 ≥ 10,000,000 后，可主动发起"轮回"。

轮回时：
✅ 保留以下内容：
   • 已解锁的涂装/皮肤/称号
   • 荣誉殿堂中的车辆（仅纪念数据，不产生加成）
   • 成就进度
   • 已获得的声望点数

❌ 重置以下内容：
   • 所有车辆（包括车库中的、生产中的）
   • 金币、零件
   • 工厂等级、产线数量
   • 科技树进度
   • 车库容量（回到 6 格）

轮回后获得：
   • 声望点数 = 累计总收入 / 1,000,000（向下取整）
   • 初始额外加成：
     - 第 1 次轮回：开局 +500 金币
     - 第 2 次轮回：开局 +1,000 金币 + 1 条额外产线
     - 第 3 次轮回：开局 +2,000 金币 + 2 条额外产线 + 品质概率 +20%
     - 每次轮回叠加，上限 10 次
```

#### 1.2.3 声望商店

声望点数用于购买永久性加成（不随轮回重置）：

| 商品 | 消耗声望 | 效果 | 限购 |
|------|----------|------|------|
| 🚀 加速器 | 5 | 全车型生产速度 +10% | 5 次 |
| 💰 财神爷 | 5 | 全车型收入 +10% | 5 次 |
| 🧬 基因改造 | 10 | 新车稀有特质概率 +5% | 3 次 |
| 🏠 空间折叠 | 15 | 车库初始容量 +2 | 3 次 |
| 🌟 星光之路 | 20 | 全车型暴击率 +5% | 3 次 |
| 🎨 轮回专属涂装 | 30 | 只有轮回玩家能用的涂装 | 1 次 |
| 🏆 轮回者称号 | 50 | 显示在名字前的金色称号 | 1 次 |
| 🔄 无尽挑战 | 100 | 解锁"无尽模式"（没有时间限制的挑战） | 1 次 |

#### 1.2.4 难度倍率

每次轮回后，游戏进程的数值会略微提高（给老玩家挑战）：

| 轮回次数 | 收入倍率 | 建造成本倍率 | 科技解锁条件倍率 |
|----------|----------|--------------|-----------------|
| 0（首次） | 1.0× | 1.0× | 1.0× |
| 1 | 1.0× | 1.1× | 1.0× |
| 2 | 1.0× | 1.2× | 1.0× |
| 3+ | 1.1× | 1.3× | 1.1× |
| 5+ | 1.2× | 1.5× | 1.2× |
| 10 | 1.5× | 2.0× | 1.5× |

**设计意图**：每次轮回≈10-15 分钟的挑战增量，轮回 10 次≈多出 2-3 小时的游戏内容。

#### 1.3.3 零件获取公式

```
零件获取 = 订单收入 × 0.01 × (1 + 0.5 × 该车是否拥有"探索发现"天赋)
```

- 每完成一个订单获得微量零件
- 进化后的轮船（豪华邮轮）额外 +100% 零件（天赋"探索发现"）
- 火箭进化后（重型火箭）零件产出 +50%（天赋"星际物流"）

| 等级 | 所需经验 | 累计经验 |
|------|----------|----------|
| 1→2 | 50 | 50 |

| 4→5 | 169 | 407 |
| 5→6 | 253 | 660 |
| 6→7 | 380 | 1,040 |
| 7→8 | 570 | 1,610 |
| 8→9 | 855 | 2,465 |
| 9→10 | 1,282 | 3,747 |

#### 1.3.5 订单收入公式

```
单次订单收入 = 车型基础售价 × (1 + 等级 × 0.05) × 品质系数 × 天赋系数

品质系数：白=1.0，蓝=1.5，金=2.0
天赋系数：普通=1.0，不同天赋各自独立
订单类型倍率：普通=1.0，长途=2.0，贵重=3.0
```

**计算示例**：
```
一辆 T4 小汽车（基础售价 800），Lv.5，蓝品质，接普通订单：
= 800 × (1 + 5 × 0.05) × 1.5 × 1.0
= 800 × 1.25 × 1.5
= 1,500 金币

如果是长途订单（×2）：
= 3,000 金币
```

#### 1.3.6 属性升级消耗

每个属性（速度/载货/耐久）独立升级，每级递增：

### 1.4 事件系统（完整设计）

#### 1.4.1 随机事件

每 60-120 秒随机触发一个事件，打破单一节奏：

| 事件 | 触发概率 | 效果 | 持续时间 |
|------|----------|------|----------|
| 🎉 **大订单** | 15% | 刷出一个高额订单（收入 ×3） | 单次 |
| 📈 **牛市** | 10% | 所有车辆售价 +50% | 30秒 |
| 📉 **熊市** | 8% | 所有车辆售价 -30% | 30秒 |
| 🔧 **零件雨** | 12% | 下一次获得零件 ×3 | 单次 |
| 🏎️ **加速光环** | 10% | 所有产线生产速度 +50% | 20秒 |
| 😴 **全员休息** | 5% | 所有车辆停止工作 10 秒 | 10秒（负面） |
| 💡 **灵感闪现** | 8% | 当前科技研究进度 +20% | 单次 |
| 🎁 **礼包掉落** | 20% | 随机给 10-50 金币 | 单次 |
| 🌟 **双倍亲密度** | 12% | 所有互动亲密度 ×2 | 60秒 |

**触发保护**：同一事件 30 秒内不重复触发。

#### 1.4.2 突发事件（与制约联动）

| 事件 | 触发条件 | 效果 | 应对方式 |
|------|----------|------|----------|
| 🏠 **车库拥挤提醒** | 使用率 > 80% | 提示"车库快满了" | 扩建/送走 |
| 😢 **老车怀念** | 一辆车累计完成 100 单 | 弹出该车战绩卡 | 无操作，仅情感 |
| 💰 **纳税日** | 每 10 分钟 | 收取当前金币 ×5% 税款 | 可提前花掉避税（策略） |

### 1.5 离线收益详细规则

```
离线时长：不超过 2 小时
离线收益效率：在线效率的 40%

离线期间：
✅ 产线继续生产（按离线前设置）
✅ 已完成的车自动出售（按当前价格）
✅ 金币和零件持续累积
❌ 车辆不获得经验
❌ 亲密度不增长（但离线期间自动消耗互动次数）
❌ 订单不刷新
❌ 随机事件不触发

离线收益弹出（上线时）：
  "你不在的 X 小时里，
   产线共生产了 N 辆车，
   总收入 🪙 XXX 金币，
   获得 ⚙️ XXX 零件
   你的车们想你了 💕"
```

## 第四章：附录

### 4.1 全文术语表

| 术语 | 定义 |
|------|------|
| Tier / T | 交通工具的等级，T1-T10 共 10 级 |
| Lv. | 单辆车的等级，1-10 级 |
| 品质 | 白→蓝→金，决定等级上限和收入倍率 |
| 性格特质 | 每辆车出厂时随机获得的被动增益 |
| 进化 | 满级+满品质后的形态蜕变 |
| 天赋 | 进化后获得的专属能力，每个车型不同 |
| 亲密度 | 与一辆车的情感羁绊值，0-100 |
| 荣誉殿堂 | 退役车辆的永久纪念展示 |
| 传承 | 退役时将 1 个特质传给新车 |
| 派单 | 手动指派车辆去执行订单 |
| 产线 | 工厂中同时生产的生产线数量 |
| 车库 | 停放已拥有车辆的设施 |
