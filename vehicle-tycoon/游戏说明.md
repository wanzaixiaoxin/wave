# 造物运输大亨 🚗 游戏说明

> **一句话**：造车 → 养车 → 进化 → 赚钱 → 造更好的车 → 养更好的车

---

## 一、游戏目标

从一个只有 100 金币的小作坊起步，造出你的第一辆独轮车。通过**派它跑订单 → 赚金币和经验 → 升级/培养 → 解锁更高级的车型**，最终造出星际飞船，成为横跨星际的运输大亨。

---

## 二、核心循环（游戏怎么运转）

游戏的核心是**一个永不停止的循环**，每一秒都在运转：

```
① 造车
   ↓
② 派车接订单 → 车跑出去（等几十秒）
   ↓
③ 订单完成 → 获得 🪙金币 + 📈经验
   ↓
④ 金币用来：造更好的车 / 升级科技 / 扩建工厂
   经验用来：给车升级 → 提升品质 → 进化
   ↓
⑤ 解锁更高级的车型（独轮车→自行车→马车→小汽车→……）
   ↓
⑥ 回到 ①（但这次造的是更高级的车，赚得更多）
```

**这个循环不需要你一直盯着看**——车跑订单的时候你可以去干别的，它会自动完成并到账。离线也会继续生产，上线时一次性领取。

## 三、各个模块的关系

```
              ┌──────────┐
              │  🏠 车库  │ ← 你所有的车都停在这里
              └─────┬────┘
                    │
       ┌────────────┼────────────┐
       ↓            ↓            ↓
   ┌──────┐   ┌────────┐   ┌────────┐
   │ 📮   │   │ 🔬    │   │ 🏭    │
   │ 订单  │   │ 科技树 │   │ 工厂   │
   └──┬───┘   └───┬────┘   └──┬─────┘
      │           │           │
      │  解锁新车 │           │ 批量造车
      │  提升效率 │           │
      └─────┬─────┘           │
            ↓                 ↓
         ┌──────────────────────┐
         │    🪙 金币 + 📈 经验   │
         └──────────────────────┘
```

### 🏠 车库 —— 核心

你的所有车辆都停放在车库中。每辆车都是**独立的个体**，有自己的：

- **等级**（1-10 级）：跑订单获得经验来升级，每级提升收入
- **品质**（白→蓝→金）：品质越高，等级上限越高，收入倍率越高
- **性格特质**：出厂时随机获得一个被动增益（勤快=速度+15%、招财=收入+10%…… 还有🔥稀有的"幸运"特质）
- **亲密度**（0-100）：通过清洗、保养、点击互动来提升，达到 80 是进化条件之一

车库有**容量上限**（初始 6 格，可扩建到 12 格），满了就不能造新车。这时你需要做选择：扩建？还是送走一辆老车？

### 📮 订单 —— 赚金币的主要方式

有三种订单自动刷新：

| 订单类型 | 收益 | 要求 |
|----------|------|------|
| 📮 普通配送 | 基础价 × 等级加成 | 任意空闲车可接 |
| 🏔️ 长途运输 | ×2，经验更多 | 需要耐久度 ≥ 3 |
| 💎 贵重物品 | ×3 | 需要蓝品质以上 |

派车接单后，车会出去跑一段时间（30-60 秒），回来时带回来金币和经验。**订单的金额根据车的等级、品质、特质动态计算**，所以养好的车赚得更多。

### 🔬 科技树 —— 解锁新车型

科技树是一条**单线**，共 5 级：

```
L1 基础机械 ─→ L2 内燃机 ─→ L3 自动化产线 ─→ L4 全球供应链 ─→ L5 星际物流
     │              │              │                │               │
  解锁品质系统   解锁小汽车/卡车  解锁火车/轮船    解锁飞机/火箭   解锁星际飞船
                  品质上限+1      生产速度+25%     产线+1          全厂收入+50%
```

每级科技都有**前置条件**（比如需要生产一定数量的特定车型），满足后才能花费金币和零件来研究。

### 🏭 工厂 —— 批量造车

工厂可以**批量生产车辆**，不用手动一辆辆造。生产好的车自动出售换成金币。产线数量随工厂升级而增加（从 1 条到最多 5 条）。

工厂和手动造车的区别：
- **手动造车**：立即消耗资源，车直接进车库。适合造你要养的车
- **工厂生产**：花时间造，造好自动卖钱。适合批量赚钱

### 🏆 成就 —— 里程碑

完成特定目标解锁成就，获得金币/零件奖励和称号。一共 7 个成就，涵盖造车数量、进化、亲密度、收集等维度。

## 四、养成一辆车的完整流程

```
造车 → 出厂随机获得一个性格特质
  ↓
给它起个名字 ← 系统会提示你命名
  ↓
派它跑订单 → 获得经验 + 金币
  ↓
经验够了 → 升级 → 收入提升
  ↓
攒够金币和零件 → 提升品质（白→蓝→金）
  ↓
属性升级 → 点速度/载货/耐久（每级消耗递增）
  ↓
每天互动 → 清洗、保养、点击 → 提升亲密度
  ↓
当它达到 Lv.10 + 传说品质 + 亲密度 ≥ 80
  ↓
🌟 进化！→ 形态蜕变（独轮车→涡轮独轮车）
    收入翻倍 + 获得专属天赋 + 等级上限突破到 15
```

每辆车的**进化天赋不同**，所以不同车型有不同的战略价值：

| 车型 | 进化天赋 | 适合 |
|------|----------|------|
| 独轮车 | 订单速度 +20% | 快速周转 |
| 自行车 | 可连续接 2 单 | 翻倍效率 |
| 马车 | 高品质订单优先 | 稳定高收入 |
| 小汽车 | 短途订单 ×2 | 爆发型收入 |
| 卡车 | 单次收入 +50% | 批量收益 |
| 火车 | 同型每多 1 辆 +5% | 集群效应 |
| 轮船 | 每次额外得零件 | 资源型 |
| 飞机 | 订单刷新 +30% | 辅助全厂 |
| 火箭 | 零件产出 +50% | 生产资料 |
| 星际飞船 | 全车型收入 +15% | 终局光环 |

| 资源 | 用途 | 怎么获得 |
|------|------|----------|
| 🪙 金币 | 造车、升级、研究科技、扩建 | 跑订单、工厂生产 |

## 六、小技巧

1. **前期优先升级一辆车到传说品质**——传说品质的车收入是白板的 2 倍，更快积累第一桶金
2. **别把所有车都留着**——车库满了就得做选择。放进荣誉殿堂的永久纪念，拆解的能回收零件
3. **品质比等级重要**——白板车升到 Lv.10 也不如蓝品质 Lv.8 赚得多
4. **零件要省着用**——前期零件紧缺，优先给主力车升品质
5. **离线也会赚钱**——产线一直在生产，上线时一次性领取。放心关掉游戏
